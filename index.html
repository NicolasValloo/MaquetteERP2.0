Note technique Tous les modules partagent un store commun (localStorage, clés préfixées gaa_*). Les données créées dans un module sont immédiatement disponibles dans les autres — par exemple, un projet créé dans le portefeuille apparaît dans le dashboard, les phases MOP, la facturation, etc. Prototype statique, aucune communication réseau. Stack cible : Django + htmx + PostgreSQL.
